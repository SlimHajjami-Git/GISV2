# GIS Frontend - FleetTrack Pro

Angular 21+ frontend for the GIS_V2 fleet management platform.

## Features

- **Monitoring** - Real-time vehicle tracking with Leaflet maps
- **Dashboard** - Fleet overview and statistics
- **Vehicles** - Vehicle management
- **Employees/Drivers** - Driver management
- **Geofences** - Geographic zones
- **Reports** - Analytics and reporting
- **GPS Devices** - Device management

## Development

```bash
npm install
npm start
```

Open http://localhost:4200

## Production Build

```bash
npm run build -- --configuration=production
```

## Docker

```bash
docker compose up frontend
```

## Tech Stack

- Angular 21
- Leaflet (maps)
- Chart.js (charts)
- RxJS
